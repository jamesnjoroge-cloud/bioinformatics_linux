# Rosalind Bioinformatics Solutions

Personal solutions to bioinformatics problems from [Rosalind](https://rosalind.info), 
written in Python while self-teaching bioinformatics alongside a BSc in Biochemistry 
at Egerton University.

## Topics covered
- Sequence parsing (FASTA)
- Dynamic programming (Longest Common Subsequence)
- Motif finding and regular expressions
- Reading frame / ORF detection
- Basic population genetics modeling

## Tools used
- Python 3
- Biopython
- requests (for UniProt API queries)

## Structure
Each file corresponds to one solved Rosalind problem, named after its problem ID 
(e.g. `FIBD.py` = Mortal Fibonacci Rabbits).
